Ordered List of HTML tags

  1. HTML:

    The root element that contains all other HTML elements on the page. It defines the document as an HTML document.

  2. head

    Contains meta-information about the HTML document, such as the title, character set, and links to external resources like stylesheets and scripts.

  3. title

    Sets the title of the HTML document, which is displayed in the browser's title bar or tab.

  4. meta

    Provides metadata about the HTML document, including character encoding, authorship, and viewport settings for responsive design.

  5. link

    Links to external resources, such as CSS stylesheets, to style the HTML document.

  6. style

    Contains CSS rules to define the appearance and layout of HTML elements within the document.

  7. script

    Used to embed or reference JavaScript code within the HTML document for interactivity and dynamic behavior.

  8. body

    Contains the visible content of the web page, including text, images, links, and other elements.

  9. h1 to h6

    Heading tags used to define six levels of headings, with being the highest level and the lowest.

  10. p

    Defines a paragraph of text.

  11. a

    Creates hyperlinks to other web pages or resources.

  12. img

    Embeds images in the document.

  13. ul

    Defines an unordered (bulleted) list.

  14. ol

    Defines an ordered (numbered) list.

  15. li

    Represents a list item within an ordered or unordered list.

  16. div

    A generic container used for grouping and styling elements.

  17. span

    A generic inline container used for styling specific portions of text.

  18. form

    Creates a form to collect user input, which can be processed on the server.

  19. input

    Defines an input field within a form for various types of user input.

  20. button

    Creates a clickable button that can trigger actions or submit a form.
